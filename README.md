---

## Quick start

**Read the guide online:** [bastis-claude-guide.pages.dev](https://bastis-claude-guide.pages.dev)

**Auto-setup your Claude environment:**
1. Open Claude desktop app
2. Copy the contents of [`claude-bootstrap.md`](./claude-bootstrap.md)
3. Paste into a fresh chat
4. Claude walks you through the full setup interactively

## What the guide covers

- Setup — desktop app, Obsidian vault, voice input
- Web app vs desktop app — what each can and can't do
- Choosing a model — Opus, Sonnet, Haiku and when to use each
- Chat and planning — how to use Claude as a planning partner
- Research workflow — the right order and how to direct it
- File and project setup — letting Claude scaffold your structure
- Token management — how not to waste credits
- Skills and repositories — finding, installing, and building skills
- MCPs and connectors — Gmail, Drive, HubSpot, GitHub and more
- Claude Code — command-line access to your codebase
- Tips and habits — what separates average and great results
